# clementine

E-commerce platform running a noir-based web server with tools for:
* Publishing products
* Managing orders
* Designing templates
* Learning from customers
* Offering coupons
* Capturing mobile payments (iOS app)

## Usage
Using leiningen 2.x.:

    $DATABASE_URL=your_url_connection_str lein run
